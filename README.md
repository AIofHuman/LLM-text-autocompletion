# LLM-text-autocompletion

A project to compare a lightweight LSTM model for tweet autocompletion generation against a pretrained model based on transformers. The goal is to compare the two models using the ROUGE metric and provide recommendations for use on mobile devices.

How to run
1. Clone the repository to disk using the command: 
- `git clone link_to_this_repo`


2. Create and activate a virtual runtime environment:
- `python -m venv .venv`
- `source .venv/bin/activate`


3. Install dependencies
- `pip install -r requirements.txt`


4. Run the notebook solution.ipynb
